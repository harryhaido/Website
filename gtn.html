Guest The Number

The "Guess The Number" project is a small game where the program generates a random number and provides hints to help the player guess the correct number. It's a fun and interactive way to test your guessing skills and challenge yourself.

The project Guess The Number is a small game in that the program will randomly generate a number and provide hints for the gamer to guess the correct number. After each incorrect answer, the program will hint according to the provided answer. For the project, I have designed 6 algorithms using math logic, such as the number of correct digits, and number of correct positions to state hints to players.
